## Simple Interest Calculator

A calculator that calculates simple interest given principal, annual rate of interest and time period in years.

```
Input:
    p, principal amount
    t, time period in years
    r, annual rate of interest
Output
    simple interest = p*t*r
```

## Repository Structure

* `simple-interest.sh` — A Bash script for calculating simple interest.
* `compound_interest.py` — A Python script for calculating compound interest.

## How to Run

**Calculate Simple Interest (Bash):**

```bash
bash simple-interest.sh
````

**Calculate Compound Interest (Python):**

```bash
python3 compound_interest.py
```
